# DS4420-Final-Project

## Vacation Destination Reccomendation 

Authors:
Kenneth Aldridge
Andy Sun
Esha Parikh

Our project is using machine learning to predict and recommend vacation/travel destinations. We are going to use 3 machine learning methods to approach this problem, an Item-Item Collaborative Filtering, a Bayesian model, and a Neural Network. For the project, our initial dataset is from Kaggle, and contains travel information for 560 different vacation destination cities. There are many features for each city, such as region, country, budget level, climate info, and ratings for several categories.

Dataset: https://www.kaggle.com/datasets/furkanima/worldwide-travel-cities-ratings-and-climate

Files:
* proof_of_concept.ipynb: Initial loading of dataset. Data is cleaned, ratings for cities are taken out, and first iteration of an Item-Item CF
* travel_data.csv: Dataset from Kaggle containing the travel information for various cities
